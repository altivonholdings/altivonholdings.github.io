# Altivon Holdings – Official Website

A production-grade, single-file HTML website for Altivon Holdings with full English ↔ Hindi translation toggle.

## 🚀 How to Use

### Option 1: Open Directly
Just open `index.html` in any browser. No server or build step needed.

### Option 2: Deploy to Vercel / Netlify
1. Create a new project on [Vercel](https://vercel.com) or [Netlify](https://netlify.com)
2. Upload the `index.html` file (or connect your GitHub repo)
3. Done — your site is live!

### Option 3: Deploy to GitHub Pages
1. Push this folder to a GitHub repository
2. Go to Settings → Pages → Source: main branch / root
3. Your site will be live at `https://yourusername.github.io/altivon-website`

## ✏️ Customization

### Update Contact Info
Search for `+91 98765 43210` and replace with your actual phone number.
Also update `info@altivon.in` and `support@altivon.in` with real emails.

### Update WhatsApp Link
Search for `wa.me/919876543210` and replace `919876543210` with your WhatsApp number (country code + number, no spaces).

### Update Google Maps
Replace the `src` of the `<iframe>` in the contact section with your actual Google Maps embed URL.

### Change Colors
Edit the CSS variables at the top of the `<style>` block:
```css
--primary: #0f4c81;   /* Main blue */
--accent: #f97316;    /* Orange accent */
```
