Project Overview

Client: EstateIn, a forward‚Äëthinking real estate agency serving luxury and mid‚Äëmarket properties across multiple cities.

Challenge: Agents needed a way to showcase properties with high‚Äëquality media, while buyers wanted immersive experiences. The old site was static, lacked mobile optimization, and had no lead tracking.

Solution: We developed a responsive property listing platform with advanced filters, Matterport virtual tours, and a private agent dashboard for managing listings and leads. Built with React for the frontend and Node.js on the backend, with MongoDB for flexible property data.

Advanced Property Listings

Buyers can filter properties by price, location, bedrooms, amenities, and more. Each listing includes high-res photos, floor plans, neighborhood info, and contact forms. The interface is fully responsive and works on all devices.

  • Instant filtering with React state
  • Map integration (Google Maps) to show location
  • Favorites and share functionality

Matterport Virtual Tours

We integrated Matterport‚Äôs API to embed 3D tours directly on property pages. Buyers can navigate through rooms, zoom in, and get a realistic feel of the space without visiting in person. Tours load quickly and are optimized for mobile.

Agent Dashboard & CRM

Agents have a private area to add/edit listings, view lead activity (which properties a lead viewed, time spent), and communicate with potential buyers. The dashboard provides analytics on listing performance and inquiry trends.

UX Research & Interviews

We interviewed agents and home buyers to understand pain points. Key insights: agents wanted easy listing management, buyers wanted immersive previews and accurate filters.

Prototyping & Wireframing

Created low‚Äëfidelity wireframes for the public site and agent dashboard. After client feedback, we built high‚Äëfidelity prototypes in Figma, focusing on simplicity and conversion.

Development (React + Node)

We built the frontend with React (Create React App) and the backend with Node.js/Express. MongoDB was chosen for its flexibility with property attributes. Integrated Matterport API and Google Maps.

Testing with Agents

Five agents tested the dashboard for a week. We collected feedback and made iterative improvements, especially around the lead tracking features.

Deployment & Training

Deployed on AWS with auto-scaling. Trained the EstateIn team on using the dashboard and managing listings. Provided documentation and ongoing support.
