## 🎨 Customization Guide

### Changing Colors
Each design has a color scheme defined in its CSS file:
```css
:root {
  --primary: #3b82f6;    /* Main brand color */
  --secondary: #8b5cf6;  /* Accent color */
  --accent: #10b981;     /* Highlight color */
}
```

### Changing Fonts
Fonts are loaded from Google Fonts in the HTML:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">
```

### Modifying Layout
- Grid layouts use CSS Grid
- Flexible sections use Flexbox
- All responsive with media queries
